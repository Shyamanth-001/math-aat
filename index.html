Math AAT

Double Integration

Syntax: integrate(function, (variable1, lower, upper), (variable2, lower, upper))

Example 1:

Evaluate: ∬ e^(x+y) dx dy with x from 0 to ln(y) and y from 1 to ln(8)

from sympy import *
x, y = symbols("x y")
f = exp(x + y)
I = integrate(f, (x, 0, log(y)), (y, 1, log(8)))
print(I)  # Output: -16 + E + 8*log(8)

Program to Compute Area (Cartesian)

Example:

Find area bounded by curves: x = y² and x = √y, with y from 0 to 1

from sympy import *
x, y = symbols("x y")
f = 1
I = integrate(f, (x, y**2, sqrt(y)), (y, 0, 1))
print("Area =", I)  # Output: Area = 1/3

Area in Polar Coordinates

Find area of cardioid: r = a(1 + cosθ)

from sympy import *
r, t, a = symbols("r t a")
f = r
I1 = 2 * integrate(f, (r, 0, a*(1 + cos(t))), (t, 0, pi))
pprint(I1)  # Output: (3 * pi * a^2) / 2

Triple Integration

Syntax: integrate(function, (x, a, b), (y, c, d), (z, e, f))

from sympy import *
x, y, z, a, b, c = symbols('x y z a b c')
f = x**2 + y**2 + z**2
I = integrate(f, (z, 0, c), (y, 0, b), (x, 0, a))
display(simplify(I))

Gradient of Scalar Field

Function: f = x²y + z at (1, 2, 3)

from sympy.vector import *
C = CoordSys3D(' ')
x, y, z = C.x, C.y, C.z
f = x**2 * y + z
G = gradient(f)
display('Gradient of f=', G)
G1 = G.subs({x:1, y:2, z:3}).evalf()
display('Gradient at (1,2,3)=', G1)

Divergence of Vector Field

Vector: F = 2x²z î - xyz² ĵ + 3yz² k̂ at (1, 1, 1)

from sympy.vector import *
C= CoordSys3D (' ')
x,y,z = C.x, C.y,C.z
i,j,k=C.i,C.j,C.k
F=(2*x**2*z)*i-(x*y**2*z)*j+(3*y*z**2)*k
G= divergence(F)
display('div F=',G)
G1=G.subs({x:1,y:1,z:1}).evalf()
display('Div F at (1,1,1) is ', G1)

if G==0:
   print("The  given vector is solendial")

   print("The  given vector is not solendial")

Irrotational Vector Field Check

Vector: F = (6xy + z³) î + (3x² − z) ĵ + (3xz² − y) k̂

C= CoordSys3D (' ')
